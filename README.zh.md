# 雙媒體同步播放器

純前端、零依賴的雙媒體同步比對播放器。檔案只在瀏覽器本機處理，不會上傳。

> English version: [README.md](README.md)

## 功能

- **影格級同步** — frame-index master clock 引擎，播放、暫停、逐格、拖動時間軸時，兩支影片永遠定位在同一個 master 時間（右側可加毫秒級偏移），暫停必定停在同一格，無漂移。
- **影片與圖片** — 支援 MP4、MOV、AVI、MKV、WebM 及 JPG、PNG、GIF、WebP、BMP。
- **四種檢視模式** — 雙視圖、只看左、只看右、重疊比較（可拖動的中央分隔線，Shift+click 瞬移）。
- **逐格比對** — 自動從 MP4/MOV 容器（mdhd/stts）解析精確 FPS（含 29.97/23.976），支援手動覆寫。
- **A-B 循環** — 標記入出點，對兩支影片同步循環播放指定片段。
- **倍速調整** — 0.25× 至 2× 播放速度。
- **縮放拖移** — 滾輪以游標為中心縮放，放大後可拖移，左右完全同步。
- **截圖** — 一鍵輸出三張 PNG（左右比較圖、左圖、右圖），含檔名標籤與時間戳。
- **色彩校正** — 自動偵測全範圍 (full-range) 與 BT.709 gamma shift 色偏並校正（macOS/QuickTime 淡化問題），截圖逐像素精確處理。
- **中／英介面** — 右上角 🌐 按鈕即時切換語言。
- **設定記憶** — 倍速、音量、偏移、檢視模式、色彩模式、語言等存於 localStorage。

## 快捷鍵

| 按鍵 | 功能 |
|------|------|
| `Space` | 播放／暫停 |
| `←` / `→` | 逐格 -1 / +1 |
| `1` `2` `3` `4` | 倍速 0.5× / 1× / 1.5× / 2× |
| `+` / `-` / `0` | 放大 / 縮小 / 重置縮放 |
| `R` | 重置縮放與位置 |
| `V` | 切換檢視模式 |
| `M` | 靜音 |
| `F` | 全螢幕 |
| `S` | 截圖 |
| `X` | 左右對調 |
| `C` | 切換色彩校正 |
| `A` / `B` / `L` | 設定 A 點 / 設定 B 點 / 切換 A-B 循環 |

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，不需安裝或建置。

或起本機伺服器：

```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000
```

## 專案結構

```
├── index.html      # UI 結構
├── css/style.css   # 樣式
└── js/
    ├── dom.js      # DOM 元素引用
    ├── state.js    # 共享狀態與常數
    ├── i18n.js     # 中／英字典與語言切換
    ├── utils.js    # 時間格式、通知等工具
    ├── probe.js    # MP4 容器解析（FPS、full-range、colr atom）
    ├── color.js    # 色彩校正（仿射轉換、BT.709 gamma LUT）
    ├── engine.js   # 影格索引主時鐘同步引擎
    ├── media.js    # 檔案載入、清除、左右對調
    ├── view.js     # 檢視模式、比較滑桿、縮放拖移
    ├── snapshot.js # 截圖輸出
    ├── settings.js # localStorage 設定持久化
    └── main.js     # 事件掛載與初始化
```

## 備註

- 播放為 seek 驅動，因此無音訊輸出（主時鐘逐格 seek 兩支影片）。本工具以逐格畫面比對為目的。
- 色彩校正係數以 macOS/Safari 實測擬合，詳見 `js/color.js` 註解。
